Rachel Fenichel's 2026 master's thesis on CS Education for BVI Students

**Abstract**: Block-based programming environments are commonly used in early computer science
education, but present accessibility barriers to students with visual impairments. Researchers
have explored the accessibility of block-based programming environments from the perspective
of human-computer interaction and computer science education. The barriers can be described as
age-related, social, and technical factors. Adding keyboard navigation to the Blockly library can
address many of these barriers.
